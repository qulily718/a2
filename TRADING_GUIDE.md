# 短线稳健策略操作指南

## 📋 策略定位

### 策略性质
- **波段操作**：非日内交易（A股T+1限制）
- **短线波段**：持有3-10个交易日
- **趋势跟随**：基于中期趋势（5-20日均线）
- **稳健风格**：控制回撤、不过度追高

### 操作周期
```
当日盘后选股 → 次日开盘前计划 → 开盘条件判断 → 买入 → 持有3-10天 → 达到目标/止损卖出
```

## 🕒 操作时机建议

### 方案一：稳健型（推荐新手）

**买入时机**：
- **策略**：次日开盘后30-60分钟观察买入
- **时间**：9:45-10:15之间
- **逻辑**：
  1. 避开开盘剧烈波动（9:30-9:45）
  2. 观察开盘是否延续昨日强势
  3. 确认成交量健康
  4. 价格在分时均线上方时介入

**价格建议**：
- 参考价格 = 昨日收盘价 × (0.995 - 1.005)
- 即：在昨日收盘价±0.5%范围内买入

**分批买入策略**：
- 第一批：开盘后符合条件时（50%仓位）
- 第二批：回踩分时均线或重要支撑（30%仓位）
- 第三批：收盘前确认趋势（20%仓位）

### 方案二：积极型（适合有经验者）

**买入时机**：
- **策略**：集合竞价或开盘立即买入
- **时间**：9:25集合竞价或9:30开盘
- **前提条件**：
  1. 昨日板块整体强势
  2. 个股有明显利好催化
  3. 隔夜外盘或消息面有利
  4. 有明确的突破信号

**价格建议**：
- 集合竞价：不超过昨日收盘价2%
- 开盘买入：开盘价±1%范围内

**风险控制**：
- 设置更紧的止损（-3%）
- 单笔仓位不超过3%
- 快速止盈（+5-8%）

## 💰 买入价格决策

系统会根据以下因素自动生成买入建议：

1. **技术评分**：≥75分积极买入，65-75分谨慎买入
2. **市场状况**：正常/牛市/熊市/震荡
3. **风险收益比**：目标≥2:1
4. **价格区间**：基于昨日收盘价计算合理买入区间

### 买入信号等级

**积极买入**（评分≥75）：
- 买入区间：昨日收盘价 -1% ~ +2%
- 建议仓位：5-7%
- 持有周期：3-5天
- 目标收益：8-12%

**谨慎买入**（评分65-75）：
- 买入区间：昨日收盘价 -1.5% ~ +1%
- 建议仓位：3.5-5%
- 持有周期：5-7天
- 目标收益：5-8%

**观望**（评分<65）：
- 不建议买入，等待更好机会

## 📊 仓位管理框架

### 金字塔式仓位管理

**单只股票仓位**：
- 高风险股票：最大3-5%
- 中风险股票：最大5-7%
- 低风险股票：最大7-10%

**总仓位控制**：
- 正常市场：总仓位≤50%
- 强势市场：总仓位≤70%
- 弱势市场：总仓位≤30%

**风险控制**：
- 单笔最大风险：总资金的2%
- 单只股票最大风险：总资金的5%

## ✅ 买入前检查清单

系统会自动检查以下条件（至少满足4项）：

1. ✅ **技术评分≥65**：技术面符合要求
2. ✅ **板块强度≥60**：所属板块表现强势
3. ✅ **市场非单边下跌**：整体市场环境健康
4. ✅ **成交量健康**：有足够的流动性
5. ✅ **价格离止损位>3%**：有足够的止损空间

## 📝 完整操作流程

### 第1步：盘后准备（T日收盘后）

1. **运行系统筛选股票**
   ```bash
   python main_realtime.py
   ```

2. **阅读生成的交易计划**
   - 查看 `results/trading_plans_YYYYMMDD_HHMMSS.txt`
   - 了解每只股票的买入区间、目标价、止损位

3. **选出3-5只备选股票**
   - 优先选择评分≥70的股票
   - 分散到不同板块
   - 控制总仓位

4. **设置明日观察清单**
   - 记录每只股票的买入区间
   - 设置价格提醒
   - 准备交易计划

### 第2步：盘中执行（T+1日）

**时间轴操作指南**：

```
9:15-9:25  观察集合竞价，不操作
9:30-9:45  观察开盘走势，确认趋势
9:45-10:15 第一批买入（如符合条件）
10:30      检查持仓，第二批决策
14:30-15:00 最后决策窗口，尾盘买入
```

**关键原则**：
1. 不在开盘前15分钟冲动交易
2. 分批买入，不一次性满仓
3. 买入后立即设置止损单
4. 记录交易日志

### 第3步：持仓管理（T+2日至卖出）

**每日收盘检查**：
- 检查持仓股票表现
- 更新止损位（跟随上涨）
- 评估是否达到目标价

**卖出条件**：
1. **达到目标价**：考虑部分止盈（50%）
2. **触及止损位**：立即全部卖出
3. **持有超过10天**：重新评估，考虑卖出
4. **技术面转弱**：评分下降，考虑减仓

## ⚡ 关键建议总结

### 买入时机
- **首选**：次日开盘后30-60分钟（9:45-10:15）
- **备选**：集合竞价或开盘（需满足前提条件）

### 买入价格
- 在昨日收盘价±1-2%范围内
- 分批买入，不追高

### 仓位控制
- 单只股票：3-7%仓位
- 总仓位：≤50%（正常市场）

### 持有时间
- 3-10个交易日
- 超过10天重新评估

### 止损纪律
- 严格执行，亏损超过3-5%坚决卖出
- 买入后立即设置止损单

### 止盈策略
- 达到5-10%考虑部分止盈
- 达到目标价考虑全部止盈

### 资金管理
- 分批买入，永不满仓
- 控制单笔风险≤2%
- 分散投资到2-3个不同板块

## 📈 风险提示

1. **市场风险**：股市有风险，投资需谨慎
2. **策略风险**：本策略适合震荡和上升市场，单边下跌市场需谨慎
3. **执行风险**：严格执行止损纪律，避免情绪化交易
4. **仓位风险**：控制总仓位，避免过度集中

## 🔄 持续优化

1. **记录交易日志**：记录每笔交易的买入价、卖出价、盈亏
2. **定期复盘**：每周/每月复盘交易表现
3. **调整参数**：根据实际表现调整筛选条件和仓位管理
4. **学习改进**：不断学习市场规律，优化策略
